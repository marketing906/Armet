# Armet - Website Oficial

Sitio web corporativo moderno y minimalista para Armet - Empresa de soluciones tecnológicas innovadoras.

## 🚀 Características

- **Diseño Minimalista**: Interfaz limpia y elegante con animaciones suaves
- **Totalmente Responsivo**: Optimizado para todos los dispositivos (móvil, tablet, desktop)
- **Rendimiento Optimizado**: Carga rápida y experiencia fluida
- **Animaciones Interactivas**: Efectos visuales modernos y atractivos
- **Formulario de Contacto**: Sistema de contacto con validación en tiempo real

## 📁 Estructura del Proyecto

```
ARMET/
├── index.html          # Página principal
├── css/
│   └── styles.css      # Estilos CSS
├── js/
│   └── script.js       # JavaScript interactivo
└── README.md           # Documentación
```

## 🎨 Secciones

1. **Hero/Inicio**: Presentación principal con llamadas a la acción
2. **Servicios**: Catálogo de servicios ofrecidos
   - Desarrollo Web
   - Aplicaciones Móviles
   - Cloud & DevOps
   - Diseño UX/UI
   - Consultoría IT
   - Inteligencia Artificial
3. **Proyectos**: Casos de éxito y portfolio
4. **Nosotros**: Información sobre la empresa y estadísticas
5. **Contacto**: Formulario de contacto e información de ubicación

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3 (Variables CSS, Flexbox, Grid)
- JavaScript (ES6+)
- Google Fonts (Inter)

## 📱 Cómo Usar

1. Abre el archivo `index.html` en tu navegador web
2. O usa un servidor local:

```bash
# Con Python
python3 -m http.server 3000

# Con Node.js (http-server)
npx http-server -p 3000

# Con PHP
php -S localhost:3000
```

3. Visita `http://localhost:3000` en tu navegador
